# SMSTrans
SMSTrans APP
短信转家-通过网络转发你的短信

描述：
通过网络免费转发你的手机收到的短信到另一个手机，从此轻松出门。

简介：

不同于市场上常用的通过运营商的短信功能进行转发，该应用通过网络进行转发，完全免费，适用于不想携带多个手机，但是又不想错过验证码等短信的用户，也适用于人在国外，没有开漫游的手机留在国内的用户。从此不再错过任何短信
